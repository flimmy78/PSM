程序运行平台：WFMT-91XX型掌机
内存资源要求：512k

规约配置：Q/GDW376.1-2009,Q/GDW376.1-2012

功能支持：用于低压集抄现场调试及生产调试，含掌机系统配置、档案录入、485检测、档案下发、终端调试、无线调试6大功能块。
	系统配置：
	1、配置修改终端与掌机通信方式
	2、配置电表485检测方式
	3、配置终端地址进制
	4、配置终端、采集器、电表地址获取方式
	5、配置低压集抄组网方式
	6、查看当前软件版本
	
	档案录入：
	1、定位、添加、修改、删除、清空终端档案
	2、定位、添加、修改、删除采集器档案
	3、定位、添加、修改、删除电表档案
	4、录入或修改电表属性
	
	485检测：
	1、批量检测485表，自动识别485表通信规约、波特率并显示检测结果
	2、逐个检测495表，自动识别485表通信规约、波特率并显示检测结果
	
	档案下发
	1、批量下发、抄读、删除集中器电表档案
	2、逐个下发、抄读电表档案
	3、查询当前电表所属终端信息
	4、录入或修改电表属性
	
	终端调试
	1、设置、查询集中器常规参数、下发集中器控制命令
	2、批量抄读集中器测量点数据、批量透抄集中器下挂电表数据
	3、逐个抄读测量点数据、透抄电表数据
	4、显示批量抄读测量点数据结果

	0 ―― 无输入
	d ―― BIN 二进制编码（采用10进制输入输出）
	h ―― BCD 二-十进制编码 （采用16进制输入输出）
	i ―― IP IP地址类型 （采用ip地址方式输入输出）
	a ―― ASCII码 （采用字符型输入输出）
	e ―― 枚举型
	t ―― 时间型
	l ―― 长度型（用户名、密码长度 可变）
	n ―― number类型（表示数据包个数 可变）
	s ―― BS结构及特殊处理型
	5 ―― 特殊处理类型（对应376.1规约 数据格式05）
